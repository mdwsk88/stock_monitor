# 企业微信群公告

这份文档提供一版可直接发到企业微信群的公告文案，用于告诉群成员如何正确使用机器人。

## 推荐群公告

```text
各位，这个群里的 A 股机器人已经接入研究型分析工具，之后可以直接 @Stock expert 提问。

推荐问法：
1. @Stock expert 茅台最近怎么看？
2. @Stock expert 平安银行最近有什么利好利空？
3. @Stock expert 今天看什么票？
4. @Stock expert 今天什么方向最强？
5. @Stock expert 低空经济看哪几只？
6. @Stock expert 今天有什么风险票？
7. @Stock expert 龙泉股份为什么会出现在今天晚报里？
8. @Stock expert 宗申动力今天是不是主线共振票？

机器人更擅长回答：
- 某只股票最近的核心事件和多空判断
- 今天市场主线、热点方向
- 宏观题材和个股公告的共振机会
- 近期高风险或偏利空的股票提示
- 今天晚报里某只股票为什么上榜

使用建议：
- 尽量直接说股票名、简称或代码
- 想看个股分析，就直接问“XXX 最近怎么看”
- 想看市场机会，就直接问“今天看什么票”或“今天什么方向强”
- 想追问晚报里的某只股票，就直接问“XXX 为什么会出现在今天晚报里”
- 如果要看原始公告明细，可以明确说“把原始公告给我看看”

注意：
- 机器人给的是事件和信号分析，不是收益承诺
- 如果当前没有高价值事件，机器人会直接说明
- 机器人不处理天气、写代码、生活建议这类无关问题
- 群里提问尽量一次只问一个主题，回答会更稳定
```

## 建议发布方式

- 新群：先发“推荐群公告”
- 老群：先发“更短的群公告版本”
