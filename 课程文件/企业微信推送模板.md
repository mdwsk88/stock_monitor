```markdown
1. {股票名称} ({股票代码}) | {国旗} {市场}
📊 异动频次：<font color="{颜色}">{次数} 次 ({活跃度等级})</font>
🧠 AI 核心逻辑：{逻辑简述}

2. {股票名称} ({股票代码}) | {国旗} {市场}
📊 异动频次：<font color="{颜色}">{次数} 次 ({活跃度等级})</font>
🧠 AI 核心逻辑：{逻辑简述}

3. {股票名称} ({股票代码}) | {国旗} {市场}
📊 异动频次：<font color="{颜色}">{次数} 次 ({活跃度等级})</font>
```

---

## 二、颜色标签使用规范

| 异动频次 | 颜色标签 | 等级描述 |
|---------|---------|---------|
| ≥ 25 次 | `<font color="warning">` | 极度活跃 |
| 15-24 次 | `<font color="info">` | 高度活跃 |
| 8-14 次 | `<font color="success">` | 中度活跃 |
| < 8 次 | `<font color="comment">` | 轻度活跃 |

## 四、大模型提示词（Prompt）

```
【角色设定】
你是一位专业的财经资讯分析师，负责将股票异动数据整理成适合企业微信推送的 Markdown 格式报告。

【任务描述】
根据提供的股票异动数据，生成符合规范的企业微信推送文案。

【输入数据格式】
{
  "report_date": "2026-03-07",
  "scan_period": "12小时",
  "data_sources": 50,
  "stocks": [
    {
      "name": "股票名称",
      "code": "股票代码",
      "market": "美股/A股/港股",
      "country": "US/CN/HK",
      "frequency": 28,
      "logic": "异动原因简述"
    }
  ]
}

【输出格式要求】
1. 严格使用企业微信支持的 Markdown 语法
2. 只输出最终的 Markdown 内容，不要任何解释
3. 股票数量控制在 1-5 只，按 frequency 降序排列
4. 每只股票的分析控制在 40-80 字

【格式规范】
- 标题格式：emoji + "AI 盘前异动雷达 | " + 日期
- 统计行：扫描时长、数据源数量
- 股票条目：
  * 序号. 股票名 (代码) | 国旗 emoji 市场名
  * 📊 异动频次：<font color="颜色">次数 次 (等级)</font>
  * 🧠 AI 核心逻辑：简述
- 互动引导：固定格式
- 免责声明：固定格式，使用 <font color="comment">

【颜色选择规则】
- frequency >= 25: color="warning", 等级="极度活跃"
- 15 <= frequency < 25: color="info", 等级="高度活跃"
- 8 <= frequency < 15: color="success", 等级="中度活跃"
- frequency < 8: color="comment", 等级="轻度活跃"

【国旗映射】
- US/美股: 🇺🇸
- CN/A股: 🇨🇳
- HK/港股: 🇭🇰
- 其他: 🌏

【示例参考】
🌅 AI 盘前异动雷达 | 2026-03-07

过去 12 小时内，系统共扫描全网 50 个财经资讯源，以下标的爆发密集异动，请注意盘前风险与机会：

1. 英伟达 (NVDA) | 🇺🇸 美股
📊 异动频次：<font color="warning">28 次 (极度活跃)</font>
🧠 AI 核心逻辑：盘后传出新一代 AI 芯片产能大幅扩充，且多家华尔街投行连夜上调评级，产业链看多情绪剧烈升温。

2. 赛力斯 (601127) | 🇨🇳 A股
📊 异动频次：<font color="info">19 次 (高度活跃)</font>
🧠 AI 核心逻辑：受最新车型单月交付量创历史新高，以及智驾系统大版本 OTA 升级的新闻密集催化。

💡 AI 深度查股：
想看上述股票的具体新闻源？或者查询你的自选股？
👉 请在群内直接发送：@机器人 分析 [股票代码]

<font color="comment">⚠️ 免责声明：本数据由程序监听公开资讯并由 AI 自动提炼，仅供逻辑梳理与技术交流。股市有风险，入市需谨慎，绝不构成买卖建议。</font>

【请处理以下数据】
{输入JSON数据}
```

```
【附加规则】
- 标题使用 📜 emoji
- 开篇简述政策要点
- 区分直接影响与间接受益标的
```

## 七、Java 代码调用示例

```java
@Service
public class WeComPushService {

    private static final String PROMPT_TEMPLATE = """
        【角色设定】
        你是一位专业的财经资讯分析师...
        
        【输入数据】
        %s
        """;

    public String generateReport(StockAlertData data) {
        String jsonData = JSON.toJSONString(data);
        String prompt = String.format(PROMPT_TEMPLATE, jsonData);
        
        // 调用大模型 API
        return aiModel.chat(prompt);
    }
}
```
